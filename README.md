## *Fetching Color Schemes for You!*

ColorDog is a simple color randomizer, allowing users to get color ideas for projects.

There are five main functions:
+ **Number of Colors** - Use the dropdown menu to select how many colors will be picked by "Randomize Colors". Number can range from 3 to 10.
+ **Randomize Colors** - Click this button to generate a new batch of random colors.
+ **Invert Colors** - Click this button to invert the displayed colors. 
+ **Sort Colors** - Click this button to loosely sort the displayed colors. 
+ **Save PNG** - Click this button to save the currently displayed colors (and their hex codes) to a PNG file name "ColorDog_colorSwatch.png"
